## 1. Objective
To perform a "Black Box" discovery of the infrastructure, identifying active hosts, open ports, and running services without prior knowledge of the internal architecture.

## 2. Tools Used
| Tool | Category | Usage |
| :--- | :--- | :--- |
| **Nmap** | Network Scanner | Port discovery and service versioning |
| **Bash** | Scripting | Automation of scan commands |
| **Wireshark** | Packet Analyzer | (Optional) Verifying network traffic flow |

## 3. Execution Process

### Phase A: Host Discovery
**Goal:** Determine which IP addresses are live.
```bash
nmap -sn <TARGET_SUBNET>
# Example: nmap -sn 192.168.1.0/24
